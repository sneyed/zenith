# Getting started

1. Add Zenith as a dependency

```
npm install @sneyed/zenith
```

2. Add Zenith's CSS to your project

```jsx
// src/index.js
import "@sneyed/zenith/zenith.css";

ReactDOM.render(
    ...
);
```
