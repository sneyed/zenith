# Getting started

## Setup

1. Clone the project and install dependencies

```
git clone https://github.com/lokimckay/zenith.git
cd zenith
npm install
```

2. Compile components to `/dist` and watch for changes

```
npm run dev
```

## View components

You can view components using storybook or an example project

### Storybook

```
cd zenith/storybook
npm install
npm start
```

### Example project

```
cd zenith/examples/<example>
npm install
npm run ... (check the package.json of the example for a start command)
```
