# Avatar

Circular image that represents a user  
Commonly used on the right side of header navigation leading to account or profile settings

<Preview>
  <Story name="Default">
    <Avatar name="Loki McKay" />
  </Story>
</Preview>

| Property | Type                 | Required | Description                              | Default                           |
| -------- | -------------------- | -------- | ---------------------------------------- | --------------------------------- |
| name     | string               | yes      | Name of avatar. Used for image `alt` tag | `undefined`                       |
| src      | string               | no       | Image source                             | "https://i.imgur.com/niM85vH.png" |
| url      | string               | no       | Url to link to when avatar is clicked    | `undefined`                       |
| size     | integer              | no       | Size of avatar in pixels                 | 64                                |
| shape    | ["circle", "square"] | no       | Circular or square shape                 | "circle"                          |
