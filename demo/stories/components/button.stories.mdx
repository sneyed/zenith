| Property | Type           | Required | Description                    | Default     |
| -------- | -------------- | -------- | ------------------------------ | ----------- |
| label    | string         | no       | Text to display on button face | "Button"    |
| url      | string         | no       | Link to navigate to on click   | `undefined` |
| type     | "hero" / "tab" | no       | Different appearances          | "hero"      |
